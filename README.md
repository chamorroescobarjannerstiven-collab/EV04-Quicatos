# API de Gestión de Inventario - Quicatos

Esta API permite la administración de productos mediante servicios REST.

## Servicios Disponibles

### 1. Obtener Productos
- **Método:** `GET`
- **Ruta:** `/api/productos`
- **Descripción:** Devuelve la lista de productos registrados.

### 2. Crear Producto
- **Método:** `POST`
- **Ruta:** `/api/productos`
- **Descripción:** Registra un nuevo producto en el sistema. Recibe un JSON con los datos.

## Tecnologías
- Node.js, Express, CORS.
